Preview Utilities
=================

Utilities for SwiftUI previews.
+ Several implementations of `FormatStyle`.
+ Several Image Generators services with different isolations.
+ `FloatingCaptionModifier`, availble through the `.floatingCaption` view modifier function.
+ `DebugOverlayModifier`, available through the `.debugOverlay` view modifier function.


This project currently includes other utilities and extensions:
+ Extension inits for `Slider`, `Picker` with support for collections and views generated using formatters.
+ Extensions for `OptionSet` to identify each option through a shift value.
+ And many more utilities.

This project is a current catch-all for utilities that do not have a individual project to live on. Much of the code in here will migrate to its own package.
